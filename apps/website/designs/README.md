# Design Explorations

25 self-contained HTML design concepts created during the CogniFillz UI exploration phase. Each file is a standalone mockup showing the extension popup and website hero section in a different aesthetic.

These are reference files, not production code. The chosen design (Design 20: Neon Sign Glow) has been implemented in the actual extension and website source files.

To preview: `cd apps/website && npx serve . -l 3000` then visit `/designs/`.
